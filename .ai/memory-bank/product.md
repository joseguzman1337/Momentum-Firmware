# Momentum Firmware - Product Overview

## Project Purpose
Momentum Firmware is a custom firmware for the Flipper Zero device that extends its capabilities far beyond the official firmware. It's a direct continuation of the Xtreme firmware, built by the same developers who made that project special.

## Core Value Proposition
- **Feature-rich**: Includes all third-party features and apps that fulfill useful purposes and work correctly, plus many new groundbreaking functionalities
- **Stable**: Ensures the most stable experience by having deep understanding of the system and making all tweaks backwards- and inter-compatible
- **Customizable**: Allows tweaking of virtually everything - apps, animations, graphics, names, menus, keybinds - all on-device without complicated configuration

## Key Features

### Momentum Settings App
Powerful configuration application that lets users customize:
- **Interface**: Desktop animations, main menu, lockscreen behavior, file browser
- **Protocols**: SubGhz settings, custom frequencies (281-361, 378-481, 749-962 MHz), GPIO pin configuration
- **Misc**: Device name, XP level, screen options, RGB backlight configuration

### Asset Packs System
Revolutionary theming system allowing users to:
- Create and cycle through custom Asset Packs with button presses
- Load custom Animations, Icons, and Fonts easily
- Install packs from community website or Discord
- Configure Anims, Icons & Fonts per pack

### Enhanced Applications
- **Bad-Keyboard**: Advanced BadUSB with Bluetooth spoofing, custom USB parameters
- **BLE Spam App**: Bluetooth Low Energy spam functionality
- **FindMy Flipper**: Device location features
- **NFC Maker**: Enhanced NFC creation tools
- **Wardriver**: GPS coordinate saving for Sub-GHz signals

### Advanced Features
- File management on device (Cut, Copy, Paste, Show, New Dir)
- Extended JavaScript API with UsbDisk/Mass Storage support
- Advanced security measures (Lock on Boot, reset on false pins)
- Disk image management (Mount and view contents)
- Enhanced WiFi support with easy setup
- Extended keyboard with cursor movement and symbols
